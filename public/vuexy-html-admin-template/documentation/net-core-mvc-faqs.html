How to integrate in to my existing project?

Since Vuexy is a template and a starter project, it‚Äôs built as the starting point of your project. It cannot be simply installed and used with an existing project like a third party library.

Although using Vuexy with any existing project is still possible, it would require extra work to connect everything together. We strongly recommend you to either start your project with Vuexy, or move your project on top of it to have the best experience.

How to hide Template Customizer?

We have provided template with customizer, So user can check all the layout options and it's classes and can get idea about all the possibilities.

To hide customizer, just need to update wwwroot\js\config.js file's displayCustomizer option to false.

But before deployment, you might not required this customizer. Then you can check below option :

if (typeof TemplateCustomizer !== 'undefined') {
  window.templateCustomizer = new TemplateCustomizer({
    ...
    displayCustomizer: false,
    ...
  });
}
Make sure you clear the browser local storage In order to see the config changes in the template.
Find more config options from here. Just make sure, you update config.js file from wwwroot\js\ folder.
How to update search list?

For page searching options in navbar search box, add page link in search-vertical.json and search-horizontal.json inside wwwroot/json/ folder.
